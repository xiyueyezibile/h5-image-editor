# 添加文字功能

日期：2024-03-23

## 更改内容

1. 添加文字元素支持
   - 新增 TextNode 组件
   - 实现文字的渲染和编辑
   - 支持文字的移动、缩放和旋转
   - 支持双击编辑文字内容

2. 文字样式支持
   - 支持字体大小调整
   - 支持字体族选择
   - 支持文字颜色设置
   - 支持文字对齐方式
   - 支持垂直对齐方式

3. 交互优化
   - 支持拖拽移动文字
   - 支持缩放调整大小
   - 支持旋转文字
   - 支持双击编辑文字内容

## 技术改进

1. 组件结构优化
   - 分离文字节点组件

   - 优化代码组织

2. 类型系统完善
   - 完善 TextElement 接口

   - 优化类型检查

# 添加底部操作工具栏

日期：2024-03-23

## 更改内容

1. 新增底部工具栏组件 `BottomToolbar`
   - 支持显示选中对象的操作选项
   - 包括删除、旋转、缩放等基础操作
   - 根据选中对象类型显示不同的操作选项

2. 更新 `ImageEditor` 组件
   - 添加 `onSelect` 回调属性
   - 实现选中对象状态的向上传递
   - 统一形状类型的处理（rect/circle/line 统一映射为 shape）

3. 样式更新
   - 添加响应式布局支持
   - 优化移动端显示效果
   - 添加按钮点击反馈效果

   - 文本编辑选项
